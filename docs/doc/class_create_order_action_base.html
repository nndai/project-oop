Detailed Description

Lớp cơ sở cho hành động tạo đơn hàng, chứa các phương thức chung cho cả quản trị viên và người dùng. Cung cấp các phương thức để chọn khách hàng, tạo đơn hàng và xử lý sau khi đơn hàng được tạo.
